How to flash Realme C25s RMX3195

  • Download zip file of firmware
  • extract .ofp file using MCT_OFP_Extractor[pass evilicacelldotcom].rar (extract the rar first)
  • merge chunk images using simg2img [read here]
  • bypass USB mediatek auth [read here]
  • now you ready to flash Realme C25s RMX3195 using SP Flash Tool [read here]

flashing more than 4GB firmware might take 1-3 hours, please be patient :)

DEADBOOT after downgrading

  • open your SP Flash Tool
  • select scatter .txt file
  • check all partitions, uncheck preloader, opporeserve2
